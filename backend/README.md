# Financial Planning API

REST API for client financial planning — FastAPI + Tortoise ORM + SQLite/PostgreSQL.

## Stack

| Layer         | Technology                 |
| ------------- | -------------------------- |
| Framework     | FastAPI 0.136              |
| ORM           | Tortoise ORM 1.1           |
| Migrations    | Aerich 0.9                 |
| Validation    | Pydantic v2                |
| Auth          | JWT (python-jose) + bcrypt |
| Rate limiting | SlowAPI                    |
| Logging       | structlog (JSON/console)   |
| Testing       | pytest + pytest-asyncio    |

## Local Setup

```bash id="c22e8u"
# 1. Clone and enter the directory
git clone <repo>
cd financial-api

# 2. Create and activate the virtual environment
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Configure .env
cp .env.example .env
# Edit SECRET_KEY and ENCRYPTION_KEY

# 5. Initialize migrations
aerich init -t app.core.database.TORTOISE_ORM
aerich init-db

# 6. Create the superuser
python scripts/create_superuser.py

# 7. Start the API
uvicorn app.main:app --reload
```

## Main Endpoints

| Method | Route                  | Auth     | Description    |
| ------ | ---------------------- | -------- | -------------- |
| POST   | `/api/v1/auth/login`   | ❌        | Login          |
| POST   | `/api/v1/auth/refresh` | ❌        | Refresh token  |
| GET    | `/api/v1/clients`      | ✅        | List clients   |
| POST   | `/api/v1/clients`      | ✅        | Create client  |
| GET    | `/api/v1/clients/{id}` | ✅        | Client details |
| PATCH  | `/api/v1/clients/{id}` | ✅        | Update client  |
| DELETE | `/api/v1/clients/{id}` | 🔐 Admin | Remove client  |
| GET    | `/health`              | ❌        | Health check   |

## Security

* **SSN**: always encrypted at rest using Fernet (AES-128-CBC). Never exposed in responses.
* **Passwords**: bcrypt with automatic cost factor.
* **JWT**: access token (30min) + refresh token (7d), validated on every request.
* **CORS**: configurable via `ALLOWED_ORIGINS` in `.env`.
* **Rate limiting**: 60 req/min per IP (configurable).
* **Docs disabled in production**: `/docs`, `/redoc`, and `/openapi.json` are removed when `APP_ENV=production`.
